# Google Search Trends Analysis

This project analyzes global keyword popularity using Google Trends data via the Pytrends Python library.

## Features
- Dynamic keyword-based search analysis
- Top 15 country identification
- World map visualization
- Time-series trend analysis
- Related keyword comparison

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly
- Pytrends

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Open the notebook and run all cells.
